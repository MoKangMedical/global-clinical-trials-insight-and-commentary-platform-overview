# 全球临床试验洞察与评论平台

AI驱动的全球临床试验数据分析与评论平台

## 核心能力

- 全球临床试验数据聚合 (ClinicalTrials.gov/EUCTR)
- 趋势分析: 适应症/阶段/地域分布
- 智能评论: AI辅助临床试验评价
- 可视化仪表板

## 快速开始

    git clone https://github.com/MoKangMedical/global-clinical-trials-insight-and-commentary-platform-overview.git
    cd global-clinical-trials-insight-and-commentary-platform-overview
    pip install -r requirements.txt
    python src/main.py --drug "GLP-1"
